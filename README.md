What Is TUR
TUR = a community repository for Triaxis packages and Ritualized user repo, advanced users contribute packages beyond official base.

Inspired by Arch’s AUR, but tailored for Triaxis GNU/Linux.

Goal: allow users to submit and fetch packages not in the official repo.
