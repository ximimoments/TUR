# Katifetch (beta)

Katifetch is a fast, customizable Neofetch-like system information tool.

## Version
1.4~beta2-1

## Maintainer
kati dev <katifetchs@gmail.com>

## Description
Katifetch shows detailed system information in a colorful and modular layout.
It supports Linux, macOS, Windows (WSL), Termux, ChromeOS, Bedrock Linux and virtual machines.

## Install

```bash
sudo apt install ./katifetch_1.4.b2-1_all.deb
```
